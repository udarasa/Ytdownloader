    Content Policy

    Universal Media Manager only analyzes and downloads media from authorized sources:

    • Files you upload and personally own or have rights to.
    • Creative Commons–licensed content, with license shown before download.
    • Public-domain media.
    • Platforms that provide an official, user-authorized export/download API.

    Links to platforms that don't permit downloading are rejected during analysis. See js/analyze.js for the integration point where authorized API calls belong.
